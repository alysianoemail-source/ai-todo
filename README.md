# AI Todo — 智能待办管理

把混乱的想法倒进来，AI 自动分类、排优先级、出计划。

## 功能

- **🧠 AI 自动分类** — 输入任意文字，自动识别内容关联，动态聚类分组
- **🎯 智能优先级** — 紧急度 × 重要度 × 耗时，多因子评分排序
- **📝 长文总结** — 粘贴一整段长文，自动提取编号列表、识别可行动项
- **📋 预览确认** — AI 提取后弹窗预览，可编辑修改、勾选确认再加入
- **🔗 子任务合并** — 「风控指标A」「风控指标B」自动合并为一条父任务
- **📱 PWA 支持** — 添加到手机桌面，全屏打开，离线可用
- **☁️ 数据导出** — JSON 导出/导入，备份无忧

## 技术栈

纯前端单页应用，零依赖，零 token 消耗。

- HTML + CSS + 原生 JavaScript
- localStorage 本地存储
- Service Worker 离线缓存
- 所有 AI 逻辑在浏览器本地运行

## 使用

浏览器打开 `index.html`，或部署到任意静态服务器。

### 短内容（<100字）
输入框一行一个 todo，点「✨ 一键更新」，AI 自动分类排优先级。

### 长文（≥100字）  
粘贴整段文字，点「✨ 一键更新」，自动识别编号列表和可行动项，弹窗预览确认后加入。

## 部署

```bash
# 本地预览
python -m http.server 8765

# 或直接浏览器打开 index.html
```
