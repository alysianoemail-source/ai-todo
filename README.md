# AI Todo

一个把碎片想法自动整理成可执行待办的工具。

---

## 核心定位

AI Todo 帮你从混乱表达中提取可以执行的下一步。

它关注的是：**任务如何被捕捉，如何不让任务在出现的那一刻就丢失。**

---

## 为什么做这个

现实中的待办信息通常：

- 分散在微信、飞书、备忘录，没有统一入口
- 很多只是随手一记，或者根本没有整理成结构

更关键的是，大多数记录并不是标准任务，而是：

- 碎片想法
- 临时提醒
- 情绪吐槽
- 混合信息

这些内容很难进入传统 Todo 系统。

---

## AI Todo 做什么

把你随手写的一段"混乱表达"，转成可执行的待办事项。

你不需要整理结构，也不需要拆分任务。只需要记录原始想法。

---

## 输入是什么样

可以是任何形式：

> 张三数据还没给我，周五要汇报，北京酒店也没订，烦死了

> 记一下报销、机票、客户方案

> 下周事情很多，感觉要出差但还没准备好

特点：无结构、碎片化、可能带情绪、多信息混杂。

---

## AI 如何处理

从原始表达中提取行动项，去除无关表达，将模糊内容转为可执行任务。

**示例**

输入：
```
张三数据还没给我，周五要汇报，北京酒店也没订，烦死了
```

输出：
```
跟进张三数据
准备周五汇报
预订北京酒店
```

---

## 产品原则

**先记录，再整理**
不需要判断"值不值得记"。先捕捉，再处理。

**不要求结构化输入**
碎片句子、语音转文字、情绪表达、临时备注，都可以。

**AI 负责提炼，而不是规划**
重点不是制定计划，而是识别哪些内容是可以执行的行动。

**系统适应用户**
不是让用户学习系统，而是适配用户原本的记录方式。
